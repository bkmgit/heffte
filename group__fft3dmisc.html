Factorize the MPI ranks into a 2D grid.

Considers all possible factorizations of the total number of processors and select the one with the lowest area which heuristically reduces the number of ranks that need to communicate in each of the all-to-all operations.

int  direction_1d,
std::array< int, 2 > const  candidate_grid 
)
inline

Factorize the MPI ranks into a 2D grid with specific constraints.

The constraints satisfied by the grid will be as follow:

  • result[i] <= world.size[i], i.e., we don't use more processors than there are indexes
  • result[direction_1d] == 1, i.e., the grid is one-dimensional in the given direction_1d
  • the product of result[i] will be equal to the product of candidate_grid[0] * candidate_grid[1]
  • if possible, the candidate_grid factorization will be used with the implicit assumption that it will be best or optimal except when the fist constraint is violated
