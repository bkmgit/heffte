Detailed Description

Wrapper to cuFFT API for real-to-complex transform with shortening of the data.

Serves the same purpose of heffte::cufft_executor but only real input is accepted and only the unique (non-conjugate) coefficients are computed. All real arrays must have size of real_size() and all complex arrays must have size complex_size().
