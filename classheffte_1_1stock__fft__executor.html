Detailed Description

Wrapper around the Stock FFT API.

A single class that manages the plans and executions of stock fft so that a single API is provided for all backends. The executor operates on a box and performs 1-D FFTs for the given dimension. The class silently manages the plans and buffers needed for the different types. All input and output arrays must have size equal to the box.
